## Findings

### [important] Unnecessary `'use client'` directive — `tuition.tsx:L1`
**Code**: `'use client'` on a component with zero client-side hooks or interactivity
**Why it matters**: Adds the file to the client JS bundle unnecessarily, increasing page weight
**Fix applied**: Removed `'use client'` from both `cybersecurity/sections/tuition.tsx` and `artificial-intelligence/sections/tuition.tsx`

### [important] Loose equality comparison — `header.tsx:L117`
**Code**: `isMobileMenuOpen == true`
**Why it matters**: Uses `==` instead of strict equality; redundant boolean comparison
**Fix applied**: Changed to `isMobileMenuOpen`

### [minor] Typo in TailArk class `grid-rows-subrgid` — `upcoming-cohorts.tsx:L55`
**Code**: `className="grid-rows-subrgid row-span-2 grid gap-12"`
**Why it matters**: `grid-rows-subrgid` is not a real Tailwind utility — silently ignored. However, this typo originates from the TailArk source block (`features-carousel.tsx`). Per CLAUDE.md rules, TailArk CSS classes must not be modified.
**Fix applied**: None — left as-is per project rules

### [minor] Variable shadowing in minesweeper game over loop — `minesweeper-grid.tsx:L166`
**Code**: `for (const c of row)` shadows outer parameter `c: number`
**Why it matters**: Readability concern — inner `c` is `CellData`, outer `c` is a column number. Works correctly but confusing to read.
**Fix applied**: None — file copied verbatim from skill per instructions

### [minor] Hardcoded color `#F5F2EF` — `message-stream.tsx:L224`
**Code**: `background: 'linear-gradient(to bottom, #F5F2EF 0%, #F5F2EF 20%, transparent 100%)'`
**Why it matters**: Not tied to the design system; if the background color changes, this must be updated manually
**Fix applied**: None — user explicitly provided this hex value; CSS variable didn't resolve correctly in prior attempts

### [minor] Hardcoded color `#F5F2EF` — `tuition-comparator.tsx` (multiple lines)
**Code**: `bg-[#F5F2EF]` used in 4 places
**Why it matters**: Same as above — hardcoded outside the design system
**Fix applied**: None — user explicitly requested this exact value

### [minor] External image dependency — `platform-features.tsx:L47`
**Code**: `src="https://cdn.prod.website-files.com/...Ruth%20Burger.jpeg"`
**Why it matters**: External CDN dependency; if Webflow CDN changes or goes down, the image breaks

## Trade-offs & Remaining Risks
- **Duplicated section files**: AI and Cybersecurity share identical `tuition.tsx`, `tuition-comparator.tsx`, `faqs.tsx`, and `upcoming-cohorts.tsx` files. If tuition or FAQ content diverges, both copies must be updated. Consider extracting shared sections to a common location if programs stay aligned.
- **Unused TailArk reference pages**: `/customers`, `/pricing`, `/legal/[document]` routes exist from block installs but aren't linked anywhere. They're harmless but add build time. Could be deleted or added to `.gitignore` patterns.
- **`content/legal` empty directory**: Created solely to prevent build failure from the legal-document block. Should be removed if the legal route is deleted.
